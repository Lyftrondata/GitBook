# Olcsóbbat

### Overview

The [Lyftrondata](https://www.lyftrondata.com/) integration for [Olcsóbbat](https://www.lyftrondata.com/integration/olcsóbbat/)[ ](https://www.lyftrondata.com/integration/olcsóbbat/)provides the most natural way to access data while requiring no engineering.

* Comprehensive incremental load mechanism.
* Data from Olcsóbbat is available in real-time.&#x20;
* ANSI SQL is fully supported to query data with ease.
* Ready to query Olcsóbbat relational schema model.
